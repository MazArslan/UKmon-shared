For now, if you want to see if there's an event matching something you saw or imaged, start by looking at UKMON Live. If the event happened in the last two or three days,it should show up here. If its older than that, or if you want more details, look at the Orbits data on the Reports page for anything older. This will allow you to quickly see if we have a matching event and to see what we have worked out about it.
Note that many of our stations only upload to the Archive on a monthly basis, so it may take a while for the orbital and trajectory data to become available.


This page allows you to search the archive for specific events.

Note: only some of our cameras upload analytical information in near-realtime. So if you're looking for a very recent event, you should start by checking our live feed (link above) to see if we captured anything of interest. Then come back here to see if we have more detailed data. If its not on the live feed, then it won't be here!

To search the archive, select a date and time range. During busy showers we often capture thousands of meteors across the network, so we strongly suggest that initially you set the range quite tight! If you set it too wide then you may get a lot of data, and the search may time out.

All the data here is freely available to download. However if you use data in any research or publication, you must attribute it to the UK Meteor Network.
