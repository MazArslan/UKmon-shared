Search the Archive.

This page allows you to search the archive for specific events.

Select a date and time, and then set an uncertainty level. We suggest that initially you set this to 5 minutes, and then expand if you get no matches. If you set it too wide then you may get a lot of data, and the search may time out.

All the data here is freely available to download. However if you use data in any research or publication, you must attribute it to the UK Meteor Network.
