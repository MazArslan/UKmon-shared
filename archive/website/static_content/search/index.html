Search the Archive.

This page allows you to search the archive for specific events.

Note: only some of our cameras upload analytical information in near-realtime. Most camera only upload data on a monthly basis.

So if you're looking for a very recent event, you should start by checking our live feed (link above). If you find an event in the live feed you can also check here to see if we have more detailed data but bear in mind that most of our cameras only upload full data once a month.

To search the archive, select a date and time range. During busy showers we often capture thousands of meteors across the network, so we strongly suggest that initially you set the range quite tight! If you set it too wide then you may get a lot of data, and the search may time out.

The data here are released under the CC BY 4.0 license, so if you are using the data whether for scientific or other purposes, your must reference this web site and UKMON in your work.
